Como instalar:

1) Clone o projeto para para o seu computador
git clone <url-do-repositorio>
cd <nome-do-projeto>

2) Criar ambiente virtual específico para o projeto:
python3 -m venv venv

3) Ativar o ambiente virtual criado:
(linux)
source venv/bin/activate
(windows)
venv\Scripts\activate

4) Atualizar pip (recomendado):
pip install --upgrade pip

5) Instalar as demais dependências
pip install -r requirements.txt

Pronto. Agora, execute o arquivo main.py
